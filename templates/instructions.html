This system will automatically transcribe chords from an uploaded audio file. The recording should be of a single isolated instrument in order to work correctly. Instructions below will explain how to use each aspect of the system to get the best out of the available transcription options. Chord results can be modified and saved as required.

File Upload Settings

File types: WAV, MP3, or Ogg Vorbis.

Choosing a transciber:
Fast Fourier Transform (FFT)

Signal processing-based method.
Chord types: Major, Minor, Suspended (2nd and 4th), Diminished, Augmented.
Faster than CNN, more susceptible to noise.
No instrument-specific preference.
Evaluation: 88.45% accuracy against full audioPianoTriad dataset.

Convolutional Neural Network (CNN)

Deep learning-based AI model.
Chord types: Major, Minor.
Slower than FFT, less susceptible to noise.
Trained on piano chord data.
Evaluation: 99.45% accuracy on test set from audioPianoTriad dataset (Maj and Min chords).

Choosing auto / manual BPM:

Beats Per Minute (BPM) is used to split the track into chord blocks for transcription. If you know the BPM of the track, you should set this manually. Otherwise, select the automatic prediction box.

Transcription Page Settings

Once the file has been transcribed, you will be able to review and refine chord predictions before saving.

Use the audio player above the chord listing to play the file. Predicted chords are time-aligned and highlighted as the song plays. Any chords can be edited by double clicking on their chord box. The box colour will change to yellow and allow you to edit the chord name. Press esc to cancel the edit, or press enter to confirm it.

Choosing a Save Format:
Save as Plain Text (.txt)

Saves the chords as plain text which can be opened in programs like Notepad.

Use this if you want a simple text file of chord names similar to the interface, which can be easily viewed or printed as required.

All information in the chord boxes is saved as-is.

Save as MusicXML (.xml)

Saves the chords in MusicXML format which can be opened in programs like Musescore and Sibelius.

Use this if you want to work in other music software with sheet music. The chord names are converted to their relevant triads and placed on the treble clef with voice-leading automatically implemented.

Only chords named using the correct format will be saved, so bare this in mind if editing chords in the interface before saving. Otherwise an empty measure will be listed in the score for that bar.

Naming format: Note name + chord type (no spaces)
Valid note name format: C, D#, Gb etc.
Valid chord type format: maj, min, sus2, sus4, dim, aug.
Example valid chords: Cmaj, Ebaug, Bsus4, C#dim etc.
